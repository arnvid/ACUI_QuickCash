How to install:

1. download
2. unpack in World of Warcraft\Interface\Addons\
3. check that it's activated in Addons button

Commands:

/qc or /quickcash

QuickCash commands: /quickcash or /qc" );

show        - show the quickcash bar
hide        - hide the quickcash bar
alpha       - set background alpha, 0 = remove, 1 = filled, 0.5 = middle
borderalpha - set border alpha, 0 = remove, 1 = filled, 0.5 = middle
lock        - lock the bar
unlock      - unlock the bar
stick       - stick to mouse
